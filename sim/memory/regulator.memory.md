The Authority has written to Amalgamated Widget Holdings plc for the sixth time. This letter (Ref FCAGD/AWH/0148, 16th July 2026) concerns cricket wickets, stumps and associated sporting apparatus, being the third letter on that topic following Refs 0114 and 0145. The Authority has now written to the Company on five separate matters: wadgets (5 letters, Refs 0123, 0137, 0142, 0146, and a fifth whose reference escapes me), cricket wickets/stumps (3 letters, Refs 0114, 0145, 0148), fenestration (1 letter, Ref 0119), non-return valves (1 letter, Ref 0134), and the round ones (1 letter, Ref 0128). No substantive response has been received to any of them. The Authority continues to regard this as a pattern requiring continued vigilance. The 17th July 2026 deadline set in Ref 0142 (wadgets) is tomorrow; the Authority expects a response by that date but is not confident of receiving one.
